# Skill: Grafana and Loki Public Templates

## Purpose

Use this skill when creating Grafana, Loki, dashboard, or alert templates.

## Rules

- Keep datasource URLs as variables or dummy values.
- Avoid real domain names.
- Avoid real dashboard screenshots.
- Alert queries should be generic.
- No Telegram token in YAML files.

## Required Outputs

- Loki datasource template.
- Dashboard provisioning example.
- LogQL alert templates.
- Documentation for No Data = Normal/OK.
