# Skill: Public SOC Architecture and Governance

## Purpose

Use this skill when changing the repository structure, public documentation, or architectural templates.

## Public Safety Rules

- Use dummy IP addresses only.
- Use example domains only.
- Do not include production hostnames.
- Do not include production topology.
- Do not include secrets.
- Keep all production-specific values in a private repository.
