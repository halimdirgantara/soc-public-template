# Skill: Public Operations Documentation

## Purpose

Use this skill when editing public SOP, troubleshooting, backup, and validation docs.

## Rules

- Keep commands generic.
- Do not include internal hostnames.
- Do not include real incident details.
- Do not include screenshots with IPs.
- Use placeholder paths and explain how to adapt privately.

## Required Docs

- Installation.
- Telegram alerting.
- Troubleshooting.
- Backup and restore.
- Validation.
